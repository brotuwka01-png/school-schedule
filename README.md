# Школьное расписание

Это простой сайт для школьников, где можно хранить своё расписание. 

## Как пользоваться:
1. Перейди на вкладку для дня недели.
2. Введи предметы и задания.
3. Сайт сохраняет данные локально, так что твое расписание будет всегда доступно.

## Как установить:
1. Клонируйте репозиторий.
2. Откройте файл `index.html` в браузере.

## Примечания:
- Этот сайт использует GitHub Pages для хостинга.
